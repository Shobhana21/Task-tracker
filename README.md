# Task Tracker App

A simple web app to create, assign, and track tasks for your team.

## Features

- Create, edit, and delete tasks
- Assign tasks to team members
- Track task status (To Do, In Progress, Done)
- Filter tasks by assignee or status

## Installation

1. Clone the repository
   ```
   git clone https://github.com/your-username/task-tracker-app.git
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Start the app
   ```
   npm start
   ```

## Usage

Once the app is running, open `http://localhost:3000` in your browser to start adding and managing tasks.
